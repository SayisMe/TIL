# 실습용 테이블

### DEPT(department) 테이블, SALGRADE 테이블

1. DESC DEPT;
- DEPTNO 열 : 부서 번호
- DNAME 열 : 부서 이름
- LOC 열 : 부서가 위치한 지역 
![dept](https://user-images.githubusercontent.com/46295659/50766567-157ad080-12bd-11e9-9f8b-cb1d481d0711.JPG)

2. DESC SALGRADE;
- GRADE 열 : 급여 등급
- LOSAL 열 : 급여 등급의 최소 급여액
- HISAL 열 : 급여 등급의 최대 급여액
