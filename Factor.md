# 관계형 데이터베이스의 구성 요소

## 테이블

- 기본적으로 데이터를 2차원 표 형태로 저장, 관리
- 이 표 형태의 데이터 저장 공간을 테이블이라고 함
- 행, 열로 구성
- 관계형 데이터베이스에서 관계를 의미

### 행
- 하나의 개체를 구성하는 여러 값을 가로로
- 실무에서는 Row 라고 부른다

### 열
- 여러 개체의 공통 특성 정의 (예를 들어, 이름, 학번 등이 해당)
- 실무에서는 Column 라고 부른다
### 특별한 열을 키(Key)라고 부른다. 수많은 데이터를 구별할 수 있는 값
